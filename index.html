Cara Download & Memasang Aplikasi

Ikuti 4 langkah mudah berikut agar RakAman siap digunakan di HP Android Anda.

1

Ketuk Tombol Download

Ketuk tombol "Download Aplikasi Android (.apk)" di atas. File akan otomatis tersimpan di folder Download HP Anda.

2

Buka File .apk

Buka aplikasi File Manager atau notifikasi unduhan, lalu ketuk file RakAman-Digital Expiry Assistant.apk untuk mulai memasang.

3

Izinkan "Sumber Tidak Dikenal"

Jika muncul peringatan, ketuk Pengaturan, lalu aktifkan izin "Izinkan dari Sumber Ini" agar instalasi dapat dilanjutkan dengan aman.

4

Ketuk Install lalu Buka

Ketuk Install, tunggu proses selesai, lalu ketuk Buka. Ikon RakAman akan muncul di layar utama HP Anda.

📌 Catatan: Peringatan "Sumber Tidak Dikenal" adalah hal normal untuk aplikasi di luar Play Store. RakAman aman digunakan dan tidak memerlukan akun atau login.
